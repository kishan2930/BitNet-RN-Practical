React Native Task
Objective
You are required to implement a simple React Native application based on the provided designs for Login/Onboarding and Home Page.
________________


General Instructions
* Use the latest version of React Native and related dependencies.
* The code must be clean, modular, and well-arranged.
   * Follow best practices for folder structure (e.g., separating components, screens, services, etc.).
   * Add meaningful comments where necessary.
* Ensure the app runs smoothly on both Android and iOS platforms.
Responsiveness
* Ensure that the app design is fully responsive and works seamlessly on different screen sizes:
   * Smartphones (small and large screens)
   * Tablets
Requirements
* Splash Screen Setup
   * Implement a splash screen that displays when the app is launched.
* Login and Onboarding Flow
   * Use the design provided for Login and Onboarding screens:
      * Design Reference (Screenshot) (https://prnt.sc/aRc6TNON_dny)
      * Figma Design Link (https://www.figma.com/design/v8DmZMXl0tpLAIHNjBKTe5/Ecommerce-Mobile-App--Community-?node-id=7-426&p=f)
   * Integrate Firebase Authentication for Email/Password Sign-Up and Sign-In functionality.
      * Users should be able to create an account using an email and password.
      * Users should also be able to log in using their credentials.
   * Add Google Login functionality (using Firebase).
* Home Page Implementation
   * Use the design reference for the Home Page:
      * Design Reference (Screenshot) (https://prnt.sc/Ghjxn0cB2pxc)
      * Figma Design Link (https://www.figma.com/design/v8DmZMXl0tpLAIHNjBKTe5/Ecommerce-Mobile-App--Community-?node-id=7-426&p=f)
   * Display any static dummy data on the Home Page for demonstration purposes.
   * Implement Tab Navigation for the Home Page using @react-navigation (e.g., Bottom Tabs).
